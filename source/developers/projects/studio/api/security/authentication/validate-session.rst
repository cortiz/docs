================
Validate Session
================

Validate a user's active session and return if the session is valid or the user has been logged out and will now need
 to re-login.

--------------------
Resource Information
--------------------

+----------------------------+-------------------------------------------------------------------+
|| HTTP Verb                 || GET                                                              |
+----------------------------+-------------------------------------------------------------------+
|| URL                       || ``/api/2/security/validate-session``                             |
+----------------------------+-------------------------------------------------------------------+
|| Response Formats          || ``JSON``                                                         |
+----------------------------+-------------------------------------------------------------------+
|| Required Role             || Self                                                             |
+----------------------------+-------------------------------------------------------------------+

----------
Parameters
----------

None.

-------
Example
-------

^^^^^^^
Request
^^^^^^^

``GET /api/2/security/validate-session``

^^^^^^^^
Response
^^^^^^^^

``Status 200 OK``
