===============
Delete Document
===============

Deletes a document from the Solr index.

--------------------
Resource Information
--------------------

+----------------------------+-----------------------------------------------------+
|| HTTP Verb                 || POST                                               |
+----------------------------+-----------------------------------------------------+
|| URL                       || ``/api/2/search/delete``                           |
+----------------------------+-----------------------------------------------------+
|| Response Formats          || ``JSON``                                           |
+----------------------------+-----------------------------------------------------+

----------
Parameters
----------

+-------------------------+-------------+---------------+----------------------------------------+
|| Name                   || Type       || Required     || Description                           |
+=========================+=============+===============+========================================+
|| index_id               || String     ||              || The index ID                          |
+-------------------------+-------------+---------------+----------------------------------------+
|| site                   || String     || |checkmark|  || The site name                         |
+-------------------------+-------------+---------------+----------------------------------------+
|| id                     || String     || |checkmark|  || The document ID                       |
+-------------------------+-------------+---------------+----------------------------------------+

-------
Example
-------

^^^^^^^
Request
^^^^^^^

.. code-block:: none

  POST .../api/2/search/delete?index_id=editorial&site=editorial&id=%2Fsite%2Fwebsite%2Farticles%2F2017%2F1%2Fwomen-styles-for-winter%2Findex.xml

^^^^^^^^
Response
^^^^^^^^

``Status 200 OK``

.. code-block:: json

  {
    "message": "OK"
  }

---------
Responses
---------

+---------+-------------------------------------+------------------------------------------------+
|| Status || Location                           || Response Body                                 |
+=========+=====================================+================================================+
|| 200    || ``.../api/2/search/delete``        || See example above.                            |
+---------+-------------------------------------+------------------------------------------------+
|| 500    ||                                    || ``{ "message" : "Internal server error" }``   |
+---------+-------------------------------------+------------------------------------------------+
